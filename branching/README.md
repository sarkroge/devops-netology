# Домашнее задание «Ветвления в Git»

## Выполнено

В рамках задания были выполнены следующие действия:

1. Создан каталог `branching` с файлами `merge.sh` и `rebase.sh`.
2. Создан коммит `prepare for merge and rebase`.
3. Создана ветка `git-merge`, в которой выполнены два коммита:
   - `merge: @ instead *`
   - `merge: use shift`
4. В ветке `main` изменён файл `rebase.sh`.
5. От старого коммита `prepare for merge and rebase` создана ветка `git-rebase`.
6. В ветке `git-rebase` выполнены два коммита:
   - `git-rebase 1`
   - `git-rebase 2`
7. Ветка `git-merge` влита в `main` через `merge`.
8. Ветка `git-rebase` перенесена на актуальную `main` через `rebase`.
9. В процессе `rebase` были разрешены конфликты.
10. После `rebase` ветка `git-rebase` была отправлена в репозиторий с принудительным обновлением истории.
11. После этого ветка `git-rebase` была влита в `main` через fast-forward merge.

## Итог

В результате выполнена практика работы с:
- ветвлением;
- merge;
- rebase;
- разрешением конфликтов;
- принудительной отправкой изменений после rebase.
